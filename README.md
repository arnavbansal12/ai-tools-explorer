Project Overview:
AI Tools Explorer is a simple web application that allows users to explore different Artificial Intelligence tools.  
The main purpose of this project is to practice JavaScript concepts, API integration, and user interface design.

This website will display a list of AI tools along with basic information such as tool name, category, and description.  
Users will be able to search for tools, filter them based on category, and sort them in different ways.

The project is being developed as part of a JavaScript web development assignment.


Features:
Display AI tools data fetched from a public API  
Search tools by name  
Filter tools based on category  
Sort tools alphabetically or by other criteria  
Responsive design for mobile and desktop  
Simple and clean user interface  


Public API Used:
The project uses data from a public API that provides information about AI tools.


Technologies Used:
HTML  
CSS  
JavaScript (Fetch API, Array Methods)  


How to Run the Project:
1. Download or clone the repository  
2. Open the project folder  
3. Open the `index.html` file in any web browser  


Project Goal:
The goal of this project is to understand how to fetch real-time data from APIs and create an interactive and responsive web application using JavaScript.
